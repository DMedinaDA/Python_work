A simple command-line Python tool for converting between different units of length measurement: inches, feet, and yards.

📋 Overview
This script implements a user-friendly command-line interface that:

Prompts users for the input unit, output unit, and measurement value

Handles various input formats (case-insensitive)

Performs six different types of conversions:

Inches → Feet

Inches → Yards

Feet → Inches

Feet → Yards

Yards → Inches

Yards → Feet

Rounds results for readability  

💡 Key Takeaways
Demonstrates attention to detail in identifying and fixing subtle logical bugs

Shows understanding of data types (int vs float) and their impact on calculation precision

Highlights the importance of consistent output formatting

Serves as a practical example of:

User input handling

String manipulation (case-insensitive comparison)

Nested conditional statements

Basic arithmetic operations
